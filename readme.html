2.GA分布

每次點擊會送出事件參數有兩個，所在區塊及目前張數

命名規則(非選單)

以點擊下圖A區塊輪播的第一張為例，送出的GA為[ A_2, a_1 ]，其中第一個參數A_2代表所在區塊+總張數(2)，第二個小寫參數a_1代表所點擊到的張數;若點擊的是C區塊的第三張，則會送出[C_4, c_3]

命名規則(選單)

  1. 選單的第一參數為 L_1~8，如iPad為L_3，無第二參數(統一送出"tab")，
    所以結果為 [ L_3 , ‘tab’ ]
  2. 選單內banner，第一參數為所屬的選單，第二參數為目前點擊張數，如點擊Watch內第二張banner，會送出[ L_4 , l_2 ]

送出的數字會由程式計算，不需手動添加

首頁四小區塊

調整版型，row裡每一個div為一卡，注意各自包含的class name，col後面可接的數字以12為最大寬度上限

  • col-6: 為此欄位預設值，代表佔總寬度的一半(6/12 = 50%)
  • col-lg-3: 中間代入尺吋為斷點，代表若目前螢幕尺吋為lg(992px)以上，則套用此設定。

所以兩個class name合起來就是: 在螢幕尺吋992px以上就佔3/12(25%)，992px以下佔6/12(50%)

常用斷點
col-md: 大於768px
col-lg: 大於992px











<!---- [bn-list] ---->
<div class="bn-list container" data-class="C">
  <div class="row">
    <div class="col-6 col-lg-3">...</div>
    <div class="col-6 col-lg-3">...</div>
    <div class="col-6 col-lg-3">...</div>
    <div class="col-6 col-lg-3">...</div>
  </div>
</div>
<!----END [bn-list] ---->

版身

首頁版身為index_01~08.html，由下方js載入，若需新增則要注意

  • html裡要新增版身容器，注意id及data-class(GA代號自行命名)















<!---- [MFi] ---->
<div class="pd container" id="pd8" data-class="K">
  <div class="row">
    <!-- in index_08.html -->
  </div>
</div>
<!----END [MFi] ---->

<!---- [新的版身] ---->
<div class="pd container" id="pd9" data-class="KK">
  <div class="row">
    <!-- in index_09.html -->
  </div>
</div>
<!----END [新的版身] ---->

  • 新增一頁html，可直接複製其他版身html，但需修改頁面上所有的pd class及id等
    ex:複製index_08.html為index_09.html，並搜尋頁面中所有pd8，改為pd9

  • 回到index.html，再下方js添加load











<script type="text/javascript">

  ...
  $('#pd7 .row').load(loadURL + 'index_07.html');
  $('#pd8 .row').load(loadURL + 'index_08.html');
  //   ---- 新的版身 -----
  $('#pd9 .row').load(loadURL + 'index_09.html');

</script>
